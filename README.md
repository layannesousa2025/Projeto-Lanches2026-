# Hero Burgers - Explosão de Sabor 🍔

Um sistema web completo para uma hamburgueria, com landing page dinâmica, integração com API PHP e banco de dados MySQL.

## 🚀 Funcionalidades

- **Landing Page Moderna**: Design responsivo com animações suaves e tipografia elegante.
- **Autenticação de Usuário**: Sistema de login, cadastro e recuperação de senha.
- **Integração com API**: Front-end comunica-se com um back-end PHP para persistência de dados.
- **Área do Administrador**: Suporte para diferentes níveis de acesso (usuário e admin).
- **Seção de Destaques**: Exibição dinâmica de produtos vindo do banco de dados.
- **Avaliações**: Listagem de reviews de clientes integrada.

## 🛠️ Tecnologias Utilizadas

- **Front-end**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Back-end**: PHP 8.x.
- **Banco de Dados**: MySQL.
- **Ambiente Local**: XAMPP (Apache + MySQL).

## 📋 Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:
- [XAMPP](https://www.apachefriends.org/index.html) (ou qualquer servidor local com PHP e MySQL).

## 🔧 Instalação e Configuração

1. **Clonar/Mover o projeto**:
   - Mova a pasta `Hero-Lanches` para dentro do diretório `htdocs` do seu XAMPP (geralmente `C:\xampp\htdocs\`).

2. **Configurar o Banco de Dados**:
   - Abra o **Painel de Controle do XAMPP** e inicie o **Apache** e o **MySQL**.
   - Acesse `http://localhost/phpmyadmin/`.
   - Crie um banco de dados chamado `hero_burgers`.
   - Clique na aba **Importar**, selecione o arquivo `database.sql` na raiz do projeto e clique em **Executar**.

3. **Executar o App**:
   - Acesse no seu navegador: `http://localhost/Hero-Lanches/`
